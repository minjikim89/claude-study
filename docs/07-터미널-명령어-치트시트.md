# Mac 터미널 & Claude Code 필수 명령어 치트시트

> 터미널을 처음 접하는 Mac 유저를 위한 필수 명령어·단축키 총정리

---

## 1. 기본 이동 (Navigation)

| 명령어 | 기능 설명 |
|---|---|
| `pwd` | 현재 내가 위치한 폴더 경로 출력 (내 위치 확인) |
| `ls` | 현재 폴더에 있는 파일/폴더 목록 보기 |
| `ls -al` | 숨겨진 파일까지 포함하여 상세 목록 보기 (권한, 크기, 날짜 등) |
| `ls -lh` | 파일 크기를 사람이 읽기 쉬운 단위(KB, MB)로 표시 |
| `cd [폴더명]` | 특정 폴더 안으로 들어가기 |
| `cd ..` | 한 단계 상위 폴더(밖)로 나가기 |
| `cd ~` | 홈 디렉토리(/Users/사용자명)로 즉시 이동 |
| `cd -` | 직전에 있던 폴더로 되돌아가기 |
| `cd /` | 최상위 루트 디렉토리로 이동 |
| `open .` | 현재 폴더를 Finder에서 열기 (Mac 전용, 매우 유용) |
| `open [파일명]` | 해당 파일을 기본 앱으로 열기 |

## 2. 파일·폴더 조작 (File & Directory)

| 명령어 | 기능 설명 |
|---|---|
| `mkdir [이름]` | 새로운 폴더 만들기 |
| `mkdir -p a/b/c` | 중간 경로까지 한 번에 폴더 생성 (중첩 가능) |
| `touch [이름]` | 새로운 빈 파일 만들기 |
| `cp [원본] [대상]` | 파일 복사 |
| `cp -r [폴더] [대상]` | 폴더 전체 복사 (`-r` = 재귀적) |
| `mv [원본] [대상]` | 파일/폴더 이동 또는 이름 변경 |
| `rm [파일명]` | 파일 삭제 (휴지통 안 거치고 즉시 삭제!) |
| `rm -r [폴더명]` | 폴더와 내부 전체 삭제 (주의!) |
| `rm -i [파일명]` | 삭제 전 확인 메시지 표시 (안전한 삭제) |
| `cat [파일명]` | 파일 내용 전체를 터미널에 출력 |
| `less [파일명]` | 파일 내용을 스크롤하며 보기 (`q`로 종료) |
| `head -n 10 [파일]` | 파일 앞쪽 10줄만 보기 |
| `tail -n 10 [파일]` | 파일 끝쪽 10줄만 보기 |
| `wc -l [파일명]` | 파일의 총 줄 수 확인 |
| `diff [파일A] [파일B]` | 두 파일의 차이점 비교 |
| `which [명령어]` | 해당 명령어가 설치된 경로 확인 |

## 3. 검색 (Search)

| 명령어 | 기능 설명 |
|---|---|
| `find . -name "*.txt"` | 현재 폴더 하위에서 .txt 파일 모두 찾기 |
| `grep "검색어" [파일]` | 파일 안에서 특정 텍스트 검색 |
| `grep -r "검색어" .` | 현재 폴더 하위 전체에서 텍스트 검색 |
| `grep -i "검색어" [파일]` | 대소문자 구분 없이 검색 |
| `grep -n "검색어" [파일]` | 검색 결과에 줄 번호 표시 |

## 4. 터미널 공통 단축키 (Shell Shortcuts)

### 실행 제어

| 단축키 | 기능 설명 |
|---|---|
| `Tab` | 파일/폴더/명령어 이름 자동 완성 (가장 많이 씀!) |
| `Tab Tab` | 자동 완성 후보가 여러 개일 때 목록 표시 |
| `Ctrl + C` | 실행 중인 명령어 강제 종료 / 취소 |
| `Ctrl + Z` | 실행 중인 프로세스를 일시 중지 (백그라운드로) |
| `Ctrl + D` | 터미널 세션 종료 (exit과 동일) |
| `Ctrl + L` | 화면 깨끗하게 비우기 (`clear` 명령어와 동일) |
| `↑` / `↓` | 이전/다음에 입력했던 명령어 다시 불러오기 |
| `!!` | 직전 명령어 재실행 (예: `sudo !!` → 직전 명령을 관리자로 재실행) |
| `history` | 지금까지 입력한 명령어 기록 전체 보기 |
| `Ctrl + R` | 명령어 기록에서 검색 (키워드 입력 → 매칭되는 이전 명령어 표시) |

### 텍스트 편집 (커맨드 라인 내)

| 단축키 | 기능 설명 |
|---|---|
| `Ctrl + A` | 커서를 줄의 맨 앞으로 이동 |
| `Ctrl + E` | 커서를 줄의 맨 뒤로 이동 |
| `Ctrl + U` | 커서 앞쪽(왼쪽) 텍스트 전체 삭제 |
| `Ctrl + K` | 커서 뒤쪽(오른쪽) 텍스트 전체 삭제 |
| `Ctrl + W` | 커서 앞의 단어 하나 삭제 |
| `Ctrl + Y` | 마지막으로 삭제한 텍스트 붙여넣기 (Yank) |
| `Ctrl + T` | 커서 앞뒤 글자 서로 바꾸기 (오타 수정) |
| `Option + ←` | 단어 단위로 왼쪽 이동 (Mac 터미널/iTerm2) |
| `Option + →` | 단어 단위로 오른쪽 이동 |
| `Option + Delete` | 단어 하나 삭제 (Mac 기본 터미널) |

## 5. iTerm2 전용 단축키

> Mac 기본 터미널 대신 iTerm2를 쓰면 생산성이 크게 올라갑니다.

### 화면 분할·탭 관리

| 단축키 | 기능 설명 |
|---|---|
| `Cmd + D` | 화면 세로 분할 (좌우 멀티태스킹) |
| `Cmd + Shift + D` | 화면 가로 분할 (상하 멀티태스킹) |
| `Cmd + ]` / `Cmd + [` | 분할된 화면(패널) 사이를 이동 |
| `Cmd + Option + 방향키` | 분할 패널 크기 조절 |
| `Cmd + W` | 현재 패널/탭 닫기 |
| `Cmd + T` | 새로운 탭 열기 |
| `Cmd + 숫자(1~9)` | 해당 번호의 탭으로 즉시 이동 |
| `Cmd + ←` / `Cmd + →` | 이전/다음 탭으로 이동 |

### 편의 기능

| 단축키 | 기능 설명 |
|---|---|
| `Cmd + F` | 터미널 화면 내 텍스트 검색 |
| `Cmd + Shift + H` | 붙여넣기 히스토리 (복사했던 내용 목록) |
| `Cmd + ;` | 자동 완성 팝업 |
| `Cmd + K` | 스크롤 버퍼 포함 화면 완전 초기화 |
| `Cmd + /` | 현재 커서 위치 하이라이트 (커서 찾기) |
| `Cmd + Option + E` | 모든 탭/패널 동시 검색 (Expose) |
| `Cmd + Shift + I` | 모든 패널에 동시 입력 모드 (Broadcast) |
| `Cmd + Enter` | 전체 화면 토글 |
| `Cmd + +` / `Cmd + -` | 글자 크기 확대/축소 |
| `Cmd + 0` | 글자 크기 기본값으로 리셋 |

## 6. Claude Code 명령어·단축키

### 실행·종료

| 명령어/키 | 기능 설명 |
|---|---|
| `claude` | Claude Code 실행 |
| `claude "질문내용"` | 한 줄 질문 후 바로 답변 받기 (세션 없이) |
| `/exit` 또는 `Ctrl + C` × 2 | Claude Code 세션 종료, 터미널 복귀 |
| `Escape` | 현재 진행 중인 AI 응답 중단 |

### 세션 관리 슬래시 명령어

| 명령어 | 기능 설명 |
|---|---|
| `/stats` | 토큰 사용량 및 현재 세션 비용 확인 |
| `/cost` | 누적 사용 비용 확인 |
| `/compact` | 대화 요약 및 컨텍스트 최적화 (긴 대화 시 필수) |
| `/clear` | 현재 대화 내역 초기화 (새 대화 시작) |
| `/undo` | AI가 수행한 마지막 코드 변경 사항 되돌리기 |
| `/files` | Claude가 현재 읽고 있는 파일 목록 확인 |
| `/config` | 설정 확인 및 변경 |
| `/help` | 도움말 보기 |

### 입력 관련 단축키

| 단축키 | 기능 설명 |
|---|---|
| `Enter` | 줄 바꿈 (메시지 계속 작성) |
| `Ctrl + J` | 줄 바꿈 (대체 키) |
| `Option + Enter` | 메시지 전송 (기본 설정, 변경 가능) |
| `↑` / `↓` | 이전에 보냈던 프롬프트 불러오기 |

### 권한 응답 (파일 수정/명령 실행 시)

| 키 | 기능 설명 |
|---|---|
| `y` | 이번 한 번만 허용 |
| `n` | 거부 |
| `a` | 이 세션 동안 항상 허용 (Always) |

## 7. Git 기본 명령어 (버전 관리)

> Claude Code를 쓰면서 자연스럽게 접하게 되는 필수 Git 명령어

| 명령어 | 기능 설명 |
|---|---|
| `git status` | 변경된 파일 현황 확인 (가장 자주 사용) |
| `git add [파일]` | 특정 파일을 커밋 대기(스테이징) 상태로 |
| `git add .` | 변경된 파일 전체를 스테이징 |
| `git commit -m "메시지"` | 스테이징된 파일을 저장(커밋) |
| `git log --oneline` | 커밋 히스토리 간략하게 보기 |
| `git diff` | 수정된 내용 비교 보기 |
| `git branch` | 현재 브랜치 목록 확인 |
| `git checkout -b [이름]` | 새 브랜치 생성 후 이동 |
| `git push` | 로컬 커밋을 원격 저장소에 업로드 |
| `git pull` | 원격 저장소의 최신 변경사항 가져오기 |
| `git clone [URL]` | 원격 저장소를 내 컴퓨터에 복제 |

## 8. 시스템 확인 (System Info)

| 명령어 | 기능 설명 |
|---|---|
| `df -h` | 컴퓨터 전체 디스크 남은 용량 확인 |
| `du -sh .` | 현재 폴더가 차지하는 용량 확인 |
| `du -sh *` | 현재 폴더 내 각 항목별 용량 확인 |
| `top` | 실시간 CPU 및 메모리 사용량 감시 (`q`로 종료) |
| `htop` | top의 개선 버전 (별도 설치 필요: `brew install htop`) |
| `ps aux` | 현재 실행 중인 모든 프로세스 목록 |
| `kill [PID]` | 특정 프로세스 종료 (PID는 `ps`로 확인) |
| `whoami` | 현재 로그인된 사용자 이름 확인 |
| `sw_vers` | macOS 버전 정보 확인 |
| `uname -a` | 시스템 커널 정보 확인 |
| `uptime` | 컴퓨터 가동 시간 확인 |

## 9. 패키지 관리 (Homebrew)

> Mac에서 개발 도구를 설치할 때 사용하는 패키지 관리자

| 명령어 | 기능 설명 |
|---|---|
| `brew install [패키지]` | 패키지 설치 |
| `brew uninstall [패키지]` | 패키지 삭제 |
| `brew list` | 설치된 패키지 목록 |
| `brew update` | Homebrew 자체 업데이트 |
| `brew upgrade` | 설치된 패키지 전체 최신 버전으로 업그레이드 |
| `brew search [키워드]` | 패키지 검색 |

## 10. 권한·네트워크

| 명령어 | 기능 설명 |
|---|---|
| `sudo [명령어]` | 관리자 권한으로 실행 (비밀번호 입력 필요) |
| `chmod +x [파일]` | 파일에 실행 권한 부여 |
| `chown [사용자] [파일]` | 파일 소유자 변경 |
| `ping [주소]` | 네트워크 연결 상태 확인 (`Ctrl+C`로 중단) |
| `curl [URL]` | URL에서 데이터 가져오기 (API 테스트 등) |
| `ifconfig` | 네트워크 인터페이스 정보 (IP 주소 확인) |
