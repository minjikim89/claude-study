# Macターミナル & Claude Code 必須コマンドチートシート

> ターミナルを初めて使うMacユーザーのための必須コマンド・ショートカット総まとめ

---

## 1. 基本的な移動（Navigation）

| コマンド | 機能の説明 |
|---|---|
| `pwd` | 現在自分がいるフォルダのパスを表示（現在地の確認） |
| `ls` | 現在のフォルダにあるファイル/フォルダの一覧を表示 |
| `ls -al` | 隠しファイルも含めて詳細一覧を表示（権限、サイズ、日付など） |
| `ls -lh` | ファイルサイズを人が読みやすい単位（KB、MB）で表示 |
| `cd [フォルダ名]` | 特定のフォルダに移動 |
| `cd ..` | 1つ上の階層（外）に移動 |
| `cd ~` | ホームディレクトリ（/Users/ユーザー名）に即時移動 |
| `cd -` | 直前にいたフォルダに戻る |
| `cd /` | 最上位のルートディレクトリに移動 |
| `open .` | 現在のフォルダをFinderで開く（Mac専用、非常に便利） |
| `open [ファイル名]` | 該当ファイルをデフォルトアプリで開く |

---

## 2. ファイル・フォルダの操作（File & Directory）

| コマンド | 機能の説明 |
|---|---|
| `mkdir [名前]` | 新しいフォルダを作成 |
| `mkdir -p a/b/c` | 中間パスを含めて一度にフォルダを作成（ネスト可） |
| `touch [名前]` | 新しい空のファイルを作成 |
| `cp [元] [先]` | ファイルをコピー |
| `cp -r [フォルダ] [先]` | フォルダ全体をコピー（`-r` = 再帰的） |
| `mv [元] [先]` | ファイル/フォルダを移動または名前変更 |
| `rm [ファイル名]` | ファイルを削除（ゴミ箱を経由せず即時削除！） |
| `rm -r [フォルダ名]` | フォルダと中身をすべて削除（注意！） |
| `rm -i [ファイル名]` | 削除前に確認メッセージを表示（安全な削除） |
| `cat [ファイル名]` | ファイルの内容全体をターミナルに表示 |
| `less [ファイル名]` | ファイルの内容をスクロールしながら見る（`q` で終了） |
| `head -n 10 [ファイル]` | ファイルの先頭10行だけ表示 |
| `tail -n 10 [ファイル]` | ファイルの末尾10行だけ表示 |
| `wc -l [ファイル名]` | ファイルの総行数を確認 |
| `diff [ファイルA] [ファイルB]` | 2つのファイルの違いを比較 |
| `which [コマンド]` | 該当コマンドがインストールされているパスを確認 |

---

## 3. 検索（Search）

| コマンド | 機能の説明 |
|---|---|
| `find . -name "*.txt"` | 現在のフォルダ以下から.txtファイルをすべて検索 |
| `grep "検索語" [ファイル]` | ファイル内で特定のテキストを検索 |
| `grep -r "検索語" .` | 現在のフォルダ以下全体でテキストを検索 |
| `grep -i "検索語" [ファイル]` | 大文字小文字を区別せずに検索 |
| `grep -n "検索語" [ファイル]` | 検索結果に行番号を表示 |

## 4. ターミナル共通ショートカット（Shell Shortcuts）

### 実行制御

| ショートカット | 機能の説明 |
|---|---|
| `Tab` | ファイル/フォルダ/コマンド名を自動補完（最もよく使う！） |
| `Tab Tab` | 自動補完の候補が複数あるとき一覧を表示 |
| `Ctrl + C` | 実行中のコマンドを強制終了/キャンセル |
| `Ctrl + Z` | 実行中のプロセスを一時停止（バックグラウンドへ） |
| `Ctrl + D` | ターミナルセッションを終了（exitと同じ） |
| `Ctrl + L` | 画面をきれいにクリア（`clear` コマンドと同じ） |
| `↑` / `↓` | 以前/次に入力したコマンドを再呼び出し |
| `!!` | 直前のコマンドを再実行（例: `sudo !!` → 直前のコマンドを管理者権限で再実行） |
| `history` | これまで入力したコマンドの履歴全体を表示 |
| `Ctrl + R` | コマンド履歴から検索（キーワード入力 → マッチする以前のコマンドを表示） |

### テキスト編集（コマンドライン内）

| ショートカット | 機能の説明 |
|---|---|
| `Ctrl + A` | カーソルを行の先頭に移動 |
| `Ctrl + E` | カーソルを行の末尾に移動 |
| `Ctrl + U` | カーソルの前（左）のテキストをすべて削除 |
| `Ctrl + K` | カーソルの後（右）のテキストをすべて削除 |
| `Ctrl + W` | カーソルの前の単語を1つ削除 |
| `Ctrl + Y` | 最後に削除したテキストを貼り付け（Yank） |
| `Ctrl + T` | カーソル前後の文字を入れ替え（タイポ修正） |
| `Option + ←` | 単語単位で左に移動（Macターミナル/iTerm2） |
| `Option + →` | 単語単位で右に移動 |
| `Option + Delete` | 単語を1つ削除（Macのデフォルトターミナル） |

## 5. iTerm2専用ショートカット

> Macのデフォルトターミナルの代わりにiTerm2を使うと生産性が大きく向上します。

### 画面分割・タブ管理

| ショートカット | 機能の説明 |
|---|---|
| `Cmd + D` | 画面を縦に分割（左右のマルチタスク） |
| `Cmd + Shift + D` | 画面を横に分割（上下のマルチタスク） |
| `Cmd + ]` / `Cmd + [` | 分割した画面（パネル）間を移動 |
| `Cmd + Option + 方向キー` | 分割パネルのサイズ調整 |
| `Cmd + W` | 現在のパネル/タブを閉じる |
| `Cmd + T` | 新しいタブを開く |
| `Cmd + 数字(1〜9)` | 該当番号のタブに即時移動 |
| `Cmd + ←` / `Cmd + →` | 前/次のタブに移動 |

### 便利機能

| ショートカット | 機能の説明 |
|---|---|
| `Cmd + F` | ターミナル画面内のテキストを検索 |
| `Cmd + Shift + H` | 貼り付け履歴（コピーした内容の一覧） |
| `Cmd + ;` | 自動補完のポップアップ |
| `Cmd + K` | スクロールバッファを含め画面を完全にクリア |
| `Cmd + /` | 現在のカーソル位置をハイライト（カーソル探し） |
| `Cmd + Option + E` | すべてのタブ/パネルを同時に検索（Expose） |
| `Cmd + Shift + I` | すべてのパネルに同時入力モード（Broadcast） |
| `Cmd + Enter` | フルスクリーンのトグル |
| `Cmd + +` / `Cmd + -` | 文字サイズの拡大/縮小 |
| `Cmd + 0` | 文字サイズをデフォルトにリセット |

## 6. Claude Codeのコマンド・ショートカット

### 起動・終了

| コマンド/キー | 機能の説明 |
|---|---|
| `claude` | Claude Codeを起動 |
| `claude "質問内容"` | 1行の質問をして即座に回答を得る（セッションなし） |
| `/exit` または `Ctrl + C` × 2 | Claude Codeセッションを終了、ターミナルに戻る |
| `Escape` | 現在進行中のAIの返答を中断 |

### セッション管理スラッシュコマンド

| コマンド | 機能の説明 |
|---|---|
| `/stats` | トークン使用量と現在のセッションコストを確認 |
| `/cost` | 累積使用コストを確認 |
| `/compact` | 会話の要約とコンテキストの最適化（長い会話のときに必須） |
| `/clear` | 現在の会話履歴を初期化（新しい会話を開始） |
| `/undo` | AIが行った最後のコード変更を元に戻す |
| `/files` | Claudeが現在読んでいるファイルの一覧を確認 |
| `/config` | 設定の確認と変更 |
| `/help` | ヘルプを表示 |

### 入力関連のショートカット

| ショートカット | 機能の説明 |
|---|---|
| `Enter` | 改行（メッセージを続けて書く） |
| `Ctrl + J` | 改行（代替キー） |
| `Option + Enter` | メッセージを送信（デフォルト設定、変更可） |
| `↑` / `↓` | 以前に送ったプロンプトを呼び出す |

### 権限の応答（ファイル変更/コマンド実行時）

| キー | 機能の説明 |
|---|---|
| `y` | 今回のみ許可 |
| `n` | 拒否 |
| `a` | このセッション中は常に許可（Always） |

## 7. Gitの基本コマンド（バージョン管理）

> Claude Codeを使いながら自然に使うことになる必須のGitコマンド

| コマンド | 機能の説明 |
|---|---|
| `git status` | 変更されたファイルの状況を確認（最もよく使う） |
| `git add [ファイル]` | 特定のファイルをコミット待ち（ステージング）状態に |
| `git add .` | 変更されたファイル全体をステージング |
| `git commit -m "メッセージ"` | ステージングされたファイルを保存（コミット） |
| `git log --oneline` | コミット履歴を簡潔に表示 |
| `git diff` | 変更内容を比較表示 |
| `git branch` | 現在のブランチ一覧を確認 |
| `git checkout -b [名前]` | 新しいブランチを作成して移動 |
| `git push` | ローカルのコミットをリモートリポジトリにアップロード |
| `git pull` | リモートリポジトリの最新の変更を取得 |
| `git clone [URL]` | リモートリポジトリを自分のコンピューターに複製 |

## 8. システム確認（System Info）

| コマンド | 機能の説明 |
|---|---|
| `df -h` | コンピューター全体のディスク残量を確認 |
| `du -sh .` | 現在のフォルダが占めるサイズを確認 |
| `du -sh *` | 現在のフォルダ内の各項目ごとのサイズを確認 |
| `top` | リアルタイムのCPUおよびメモリ使用量を監視（`q` で終了） |
| `htop` | topの改良版（別途インストールが必要: `brew install htop`） |
| `ps aux` | 現在実行中のすべてのプロセス一覧 |
| `kill [PID]` | 特定のプロセスを終了（PIDは `ps` で確認） |
| `whoami` | 現在ログインしているユーザー名を確認 |
| `sw_vers` | macOSのバージョン情報を確認 |
| `uname -a` | システムのカーネル情報を確認 |
| `uptime` | コンピューターの稼働時間を確認 |

## 9. パッケージ管理（Homebrew）

> Macで開発ツールをインストールするときに使うパッケージマネージャー

| コマンド | 機能の説明 |
|---|---|
| `brew install [パッケージ]` | パッケージのインストール |
| `brew uninstall [パッケージ]` | パッケージの削除 |
| `brew list` | インストール済みパッケージの一覧 |
| `brew update` | Homebrew自体のアップデート |
| `brew upgrade` | インストール済みパッケージ全体を最新バージョンにアップグレード |
| `brew search [キーワード]` | パッケージの検索 |

## 10. 権限・ネットワーク

| コマンド | 機能の説明 |
|---|---|
| `sudo [コマンド]` | 管理者権限で実行（パスワードの入力が必要） |
| `chmod +x [ファイル]` | ファイルに実行権限を付与 |
| `chown [ユーザー] [ファイル]` | ファイルの所有者を変更 |
| `ping [アドレス]` | ネットワーク接続状態を確認（`Ctrl+C` で中断） |
| `curl [URL]` | URLからデータを取得（APIテストなど） |
| `ifconfig` | ネットワークインターフェース情報（IPアドレスの確認） |
